# Exploratory Data Analysis for Data Science & ML

Performed EDA on the sklearn diabetes dataset using Python.

## Key techniques covered:
- Missing value detection and imputation (mean & median)
- Histograms, boxplots, and correlation matrices
- Outlier detection and removal
- Log-normality testing with Shapiro-Wilk
- Feature engineering to improve linear regression RMSE

## Tools used:
Python, pandas, NumPy, scikit-learn, seaborn, Matplotlib, missingno
